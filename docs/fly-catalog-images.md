## When adding images

1. Save the photo as `assets/images/flies/<slug>.png` (kebab-case, e.g. `elk-hair-caddis.png`).
2. Register it in `src/constants/flyImages.ts` keyed by lowercase pattern name.
3. Check off the fly in this file and update the progress count at the top.
